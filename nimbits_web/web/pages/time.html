Time Service

Nimbits will give you the current time on App Engine, which can save you the trouble of using a separate time server or NTP. 

    prints the current time, in milliseconds from the unix epoch i.e 1332876426280
    /service/time

    prints the current time in json format  2012-03-27T19:27:57 +0000
    /service/time?format=json
