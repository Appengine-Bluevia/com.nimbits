Chart Services

Please read the web service primer to understand how to provide proper authentication parameters. For simplicity, authentication parameters are not shown here.

This is an overview of the chart image service, which can generate PNG format images of your data. You may also be interested in reading about creating dynamic, live charts here.

Nimbits is integrated into the Google ™ Image Chart API. It works simply by taking the URL parameters of a chart you would request from Google, and adds your point's data to it. You can use the Chart API to embed Nimbits data into web pages, emails and so on.

Before you use Nimbits Charting, try experimenting with making charts using the Chart API itself. There are many tutorials and even a wizard you can use to make charts. http://code.google.com/apis/chart/image/

The Chart API returns an Image in PNG format, generated using parameters in the URL you provide. This example is generated using the url below.

http://chart.apis.google.com/chart?cht=s&chd=t:12,87,75,41,23,96,68,71,34,9|98,60,27,34,56,79,58,74,18,76|84,23,69,81,47,94,60,93,64,54&chxt=x,y&chs=200x125&chco=000000,ffff10

Google Chart API adds the chd=t:... which represents the numbers to add to the chart. Simply change the url to nimbits, add point parameters, and remove any chd and Nimbits will add the data for you. The URL above becomes the new URL below. 

http://app.nimbits/com/service/chartapi?point=myPoint&cht=s&chxt=x,y&chs=200x125&chco=000000,ffff10

You can add the following parameters to the chart service request to tell nimbits how to format your chart.

  • point - a single point to add to the chart
  • points - comma delimited list of points i.e point1,point2 for multiple trends
  • count - a fixed number of values to add to the chart, current value and going back in time to the count
  • sd - the start date of a date range to show, in unix epoch milliseconds or a Nimbits timestamp
  • ed - the end date of a date range to show, in unix epoch milliseconds or a Nimbits timestamp

Example - live data from my temperature probe

http://cloud.nimbits.com/service/chartapi?point=TempF&email=bsautner@gmail.com&cht=lc&chs=200x100&chxt=x,y
