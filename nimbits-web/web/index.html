Nimbits is a service for connecting people, sensors and devices on the cloud. It is a free, social and open source platform for the Internet of Things.


Nimbits Server is an Operational Process Data Historian you can use to create and share data points on a public or private cloud and record your changing numeric, text, GPS, json or xml values into them. You can use Nimbits to store and share files, sensor logs, and process diagrams and receive alerts, perform calculations, generate statistics and more.
