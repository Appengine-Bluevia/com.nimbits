Nimbits is a service for connecting people and devices on the cloud. It is a free, social and open source platform for the Internet of Things.

Nimbits Server is a free and open source Operational Process Data Historian you can use to create and share data points on a public or private cloud and record your changing numeric, text based, GPS, json or xml values into them. You can use Nimbits to store and share files, sensor logs, and process diagrams and receive alerts, perform calculations, generate statistics and more. With our open source SDK and REST Web Service API, you can use nimbits as a backend for your software and micro-controller projects.
