Nimbits is a service for connecting people, sensors and devices on the cloud. It is a free, social and open source platform for the Internet of Things.


Nimbits Server is a free and open source Operational Process Data Historian you can use to create and share data points on a public or private cloud and record your changing numeric, text, GPS, json or xml values into them. You can use Nimbits to store and share files, sensor logs, and process diagrams and receive alerts, perform calculations, generate statistics and more. With our open source SDK and REST Web Service API, you can use nimbits as a backend for your software and micro-controller projects.

Login to the public Nimbits cloud now and start creating and sharing data points.

Download Nimbits server and spin up your own instance. You can run Nimbits on Google App Engine, Amazon EC2, or your own Virtual Machines.

Subscribe to our support services and let us expertly setup your instance and provide updates, monitoring and support

With nimbits, you can add give your projects the processing power of Google's infrastructure, add the intelligence of the WolframAlpha computational knowledge engine, connect to Facebook, Twitter and the Google Plus social network, and connect to thousands of data feeds from hundreds of online Nimbits Server instances.

Connect your projects to nimbits, upload diagrams and schematics, even share the 3D printer template with other nimbits users you're connected to or the world.

Embed live, interactive charts in your own web sites using javascript and free chart apis. Nimbits is integrated into the Google Chart Api. Check out the Chart Gallery to see all of the different types of charts, gauges, meters and maps you can embed nimbits data in. You can record data into the nimbits public cloud using any programing language. Check out this example using JQuery
