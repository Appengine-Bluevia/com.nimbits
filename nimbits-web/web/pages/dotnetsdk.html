Using the Nimbits SDK from .net

You can use the Nimbits SDK from a .net project. There is a utility you can use to convert Java JARS to .NET DLLs and add the use the DLL in your project.

  • Download the Java JAR File from the downloads page.
  • Next, download IKVM. IKVM is a utility that converts Java JARs to .Net Assemblies.
  • Run this command: 
    ikvmc -target:library [path to nimbits java jar]\nimbits.jar
  • You now have a nimbits_sdk.dll! You need to add it, and the dll assemblies in the IKVM Downloads BIN Directory as refrences in your .net project. 
   class Program
