Live JQuery Chart Sample

This is a complete sample that downloads a list of values from the series web service using a JQuery Get Request and the Google Chart Tool. The series service responds with JSON data, which is easy to turn into an object in Javascript. The anatomy of a json formatted recorded Value Objects is like this:

View Source 

    {"lt":0.0,"lg":0.0,"d":84.0,"t":1340827070070,"n":"some notes","dx":"some data","st":3}
  • lt: Latitude GPS Coords
  • lg: Longitude GPS Coords
  • d: the number value
  • t: The timestamp, in millisecond unix epoch format
  • n: A note. This would be the annotation in a chart and can be any string
  • dx: Data. Can be any string data, you can store xml or json in this channel
  • st: The alert state of this value as defined here.
