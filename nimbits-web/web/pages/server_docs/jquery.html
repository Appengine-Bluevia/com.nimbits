JQuery

It's easy to use JQuery to post and get data in and out of a nimbits data point using javascript. This code snippet uses an email address, access key , and the batch web service to record a random value. This example imports the jquery library, starts a timer, and begins recording a random value to the batch service.

POST Data to the Batch service

            <html>
            <head>

                <script type="text/javascript" src="/assets/js/jquery/jquery-1.7.1.js"></script>
                <script type="text/javascript">
                    var c=0;
                    var t;
                    var timer_is_on=0;

                    function timedCount()
                    {
                        var r=Math.floor(Math.random()*11)
                        $.post("http://app.nimbits.com/service/batch",
                                { email: "example@gmail.com", p1: "batch monitor", v1: r, key:"my key" },
                                function(data){

                                }
                        );


                        c=c+1;
                        t=setTimeout("timedCount()",1000);
                    }

                    function doTimer()
                    {
                        if (!timer_is_on)
                        {
                            timer_is_on=1;
                            timedCount();
                        }
                    }
                </script>
            </head>

            <body>
            <form>
                <input type="button" value="Start" onClick="doTimer()">

            </form>

            </body>
            </html>
