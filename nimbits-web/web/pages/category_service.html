Category Service

Please read the web service primer to understand how to provide proper authentication parameters. For simplicity, authentication parameters are now shown here.

com.nimbits.server.service.CategoryService

Creates new point categories on post, returns all categories belonging to a user on get or a single category.

Optionally returns points that belong to each category as a JSON collection

GET

  • includepoints : instructs nimbits to include a json string in the JsonPointCollection property of each category returned that is an array of data points belonging to that category
  • name -optional - if included, returns the single category instead of a list of all of them

POST

Create/Delete a category
  • name - the name of the new category if creating, or the category to be deleted if deleting.
  • action - options: create, delete â€“ specify if you want to create or delete the category. Delete will also delete all child points and their date. Default is create.

Sample

    Create a new category:
    /service/category?name=newcat
    Delete a category, and all of it's points along with their data
    /service/category?name=newcat&action=delete /service/category?name=newcat&includepoints=true
