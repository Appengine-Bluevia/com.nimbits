We can provide support and services for your business. We try to keep Nimbits free to get started or for those with the technical expertise to build and support their own instances. Since Nimbits Server is designed to be highly scalable, the installation and maintenance is slightly more complex than a non-scalable server. You can save time by ordering services below. Upon receipt of your payment we can install an instance of Nimbits Server for you on Google App Engine (App Engine Billing will apply to resources you consume). We can also monitor and maintain your instance with ongoing upgrades, support and analysis.

If you want to run an Instance of Nimbits Server on your own Google App Engine account, we can install and configure the entire system for you. For us to complete the services below, simply login to App Engine, select Permissions from the control panel, and add support@nimbits.com as a developer role to your account. We'll take it from there. Please allow 24-48 hours for your request to complete.
