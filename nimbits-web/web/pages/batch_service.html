com.nimbits.server.service.BatchService

A highly efficient service for loading many values at once in a single post. Accepts up to 100 new values for points in one post.

  • p1 - the name of the point
  • v1 - the new value belonging to p1.
  • t1 - optional - the timestamp as a long, unix epoch time in milliseconds. Can also be in scientific notation i.e 1.3190148e+12

Sample - records multiple values up to p100 and v100 

        /service/batch?p1=point1&v1=2.3&p2=point2&v2=3.24....

        /service/batch?p1=point1&v1=2.3&t1=1297655407000&p2=point2&v2=3.24....
