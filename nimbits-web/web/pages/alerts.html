Alerts

A point can be in one of four states

  • Normal All is well.
  • High: The last recorded number value was higher or equal to the high alert setting.
  • Low: The last recorded number value was lower or equal to the high alert setting.
  • Idle: The point has not received a new value in a set number of time, based on the idle setting.

When a point goes into an Alert State, it will attempt to notify you in ways set in the point's properties.
The following alert methods are available

  • Email
  • Instant Message
  • Facebook
  • Twitter
  • When using the portal or android app, point icons change color based on the alert state.
