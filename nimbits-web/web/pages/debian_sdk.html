sudo apt-get install nimbits

If you are using Ubuntu or Debian Linux, either workstation or server, you can use apt-get to install a nimbits. You will download a fresh copy of Nimbits Server you can deploy and the SDK.

Installing

This has been tested on Ubuntu 11.10 - please let us know if you have any issues on another version of linux.

Before installing Nimbits, please make sure your linux system has Java installed and in your path i.e:

sudo apt-get install openjdk-6-jdk

You should be able to open a terminal window and type 'java -version' and get a response.

edit the /etc/apt/sources.list file to add the Nimbits repository to the bottom: 

deb http://deb.nimbits.net/packages/ ./

at a terminal, run the following commands: 

sudo apt-get update

sudo apt-get install nimbits

At the moment, Ubuntu warns that the software you are installing can not be validated / authenticated. Please enter yes to install anyway. to get updates, just run:

sudo apt-get update

Usage
The install package puts the Nimbits Java SDK nimbits_sdk..jar in a folder in /opt/nimbits you can add nimbits_sdk.jar to your java projects or use it directly (java -jar nimbits_sdk.jar ....) but we also install a script file in /usr/bin so you can use the sdk from the command line like this: 
$nimbits point=TempF value =22.2

All command line entries require credentials: 

    -email= your email address
    -key =  your nimbits key
    -password  optional gmail password if no key is provided
    -host  optional your nimbits url. defaults to the public server on app.nimbits.com

Then an action: 

    action=read -point=pointname read a point's value
    action=record -point=pointname -value=1.234


Optional

    verbose=true

Using a Hash File

The SDK provides a way to generate a hash file to store some of your command like parameters. PLEASE NOTE: The file is generated using very simple encryption. While the information stored in this file is not readable as plain text, you should protect it as if it does.

This example will generate a hash file containing the path to a nimbits server, a google account and password: 

nimbits -genkey -out=/tmp/file.txt -url=http://mynimbits.appspot.com -email=test@example.com -password=password

You can then use the file instead of providing the above info in every command line: 

nimbits -v -point=foo -action=recordvalue -value=1.0 -i=/tmp/file.txt
Samples

Record the number 42 to a point named answerPoint 

nimbits -email=example@test.com password=secret -value=42.0 -point=answerPoint action=record

Get the current value from a point named myPoint 

nimbits -email=example@test.com key=4234234423423werqwer234523 -point=myPoint -action=read
