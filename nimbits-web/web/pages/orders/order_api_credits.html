Order API Credits

Quotas on the public cloud will go in affect on Sept 1, 2012

You can use the public Nimbits cloud with a free quota of 1000 api calls a day. Api calls are reset every day at midnight EST. If you want to read and write data beyond the free quota, you can fund your account by purchasing credits that will be used automatically when your account uses more than 1000 api calls each day. An api call costs approximately $0.00001, so a $20 USD block will provide about 2 million requests*.

You can use the Nimbits console to configure a maximum quota for each day so your account will not exceed a set amount.

*Please note that a cost of $0.00001 per api call is an approximation and costs may vary slightly depending on the nature of the request. Rates are subject to change.
